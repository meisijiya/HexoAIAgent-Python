🤖 Hexo Agent Widget 测试

这是一个测试页面，用于验证 Hexo Agent Widget 的功能。

📌 测试说明：
  • 点击右下角的小人图标打开对话窗口
  • 小人可以拖拽移动位置
  • 点击「匿名体验」登录
  • 输入消息测试对话功能
  • 刷新页面后小人位置会保持

📚 测试用例

1. 普通对话：输入「你好」测试基础对话

2. 知识库问答：输入「Redis 分布式锁怎么实现」测试 RAG

3. 搜索功能：输入「搜索最新的 AI 技术」测试搜索

4. 多轮对话：先说「我叫老江湖」，再问「我叫什么名字」

🎨 功能特性

✅ 小人图标（可拖拽）

✅ 对话弹窗（动画效果）

✅ 消息气泡（用户/助手）

✅ 流式输出（实时显示）

✅ Agent 信息显示

✅ 知识库来源显示

✅ 响应式适配

✅ 位置记忆
