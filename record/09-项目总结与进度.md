## 📖 项目概述

本项目是一个 Hexo 博客的 AI Agent 插件，支持知识库问答、外部搜索、多轮对话等功能。

**核心目标**：
- 为 Hexo 博客添加 AI 对话能力
- 支持 RAG 知识库问答
- 支持多轮对话记忆
- 支持 ReAct 工具调用

---

## 🏗️ 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| **前端** | Hexo + Chic 主题 | 静态博客 |
| **后端** | Python + FastAPI | AI 服务 |
| **数据库** | PostgreSQL + pgvector | 向量存储 |
| **缓存** | Redis | 会话记忆 |
| **LLM** | DeepSeek V3 | 对话生成 |
| **Embedding** | DashScope text-embedding-v4 | 文本向量化 |
| **部署** | Docker + Docker Compose | 容器化部署 |

## 📝 技术亮点

### 1. 混合意图识别

```
用户输入
    ↓
命令检查（/搜索、/知识库、/react）
    ↓
关键词快速判断（80% 情况）
    ↓
LLM 判断兜底（20% 情况）
    ↓
路由到对应 Agent
```

**优势**：
- 快速响应（减少 LLM 调用）
- 准确率高（关键词 + LLM）
- 可扩展（易添加新规则）

### 2. 双缓存机制

```
Embedding 请求
    ↓
本地缓存（进程级）
    ↓ 未命中
Redis 缓存（持久化）
    ↓ 未命中
DashScope API
    ↓
存入双缓存
```

**优势**：
- 响应快（本地缓存）
- 持久化（Redis 缓存）
- 节省成本（减少 API 调用）

### 3. 结构化 Prompt

```
System Prompt
    ↓
+ RAG Context（知识库检索结果）
+ History（对话历史）
+ User Input（用户输入）
    ↓
完整 Prompt
```

**优势**：
- 上下文完整
- 可维护
- 可复用

### 4. 容错处理

```
用户问题
    ↓
知识库检索
    ↓ 无结果
容错处理
    ├─ 引导补充问题
    ├─ 提供选项
    └─ LLM 参考答案
```

**优势**：
- 用户体验好
- 引导正确提问
- 避免死循环
