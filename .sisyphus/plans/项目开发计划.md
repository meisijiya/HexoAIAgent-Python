## 📋 详细任务清单

### Phase 1: Agent 智能路由优化

#### 1.1 Orchestrator 意图识别 v3
**文件**: `agent-service/app/core/orchestrator.py`

**修改内容**:
```python
# 调整优先级：知识库 > 搜索 > ReAct > 聊天
# 新增规则：
# - 只有 /搜索 命令或明确"搜索xxx"才走搜索
# - 移除时间词触发搜索的逻辑
# - ReAct 仅在"对比"/"分析"且问题较长时触发
# - 默认 fallback 到知识库
```

#### 1.2 Knowledge Agent 容错自动 fallback
**文件**: `agent-service/app/agents/knowledge_agent.py`

**修改内容**:
```python
# 当知识库无匹配时：
# 1. 自动切换到 LLM 自身知识回答
# 2. 添加免责声明："以下回答基于AI通用知识，非博客内容"
# 3. 不再弹出选项让用户手动选择
```

#### 1.3 ReAct Agent Prompt 强化
**文件**: `agent-service/app/agents/react_agent.py`

**修改内容**:
```python
# 强化 system prompt：
# - 明确要求必须调用工具
# - 添加工具调用示例
# - 禁止直接编造答案
```

#### 1.4 Error Handler 自动判断逻辑
**文件**: `agent-service/app/agents/error_handler.py`

**修改内容**:
```python
# 自动判断错误类型：
# - 知识库无匹配 → fallback 到 LLM
# - LLM 超时 → 重试或返回友好提示
# - 网络错误 → 提示用户稍后重试
```

```nginx

    server_name api.your-blog.com;

    location /api/ {
        proxy_pass http://2gb-server:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

        proxy_pass http://localhost:8080;
```
